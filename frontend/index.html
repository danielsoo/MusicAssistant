How it works

Upload or record any audio and our AI pipeline separates it into individual instrument stems, detects chord progressions, identifies the musical key, and generates sheet music.

1
Upload or Record an MP3, WAV, FLAC, OGG, or M4A file
2
Demucs AI separates vocals, drums, bass, guitar, piano & other into individual audio files
3
Basic Pitch analyzes each stem — detecting chords, BPM, and musical key using Krumhansl-Schmuckler
4
Edit & Arrange — correct chords, listen to stems, view sheet music, and convert modes

API Reference

The backend is a FastAPI server running locally at http://localhost:8000.

Auth
POST/auth/register
Create account. Body: {username, password, full_name?, age?, email?}. Returns token.
POST/auth/login
Sign in. Body: {identifier, password}. Returns token.
POST/auth/google
Google login. Body: {credential}. Returns token.
Analysis
POST/analyze
Upload audio file. Returns job_id. Add Authorization: Bearer <token> to save results.
GET/status/{job_id}
Poll: queued → separating → analyzing → done
GET/result/{job_id}
Full result: stems, chords, BPM, key, ABC notation.
Library
GET/my-jobs
List saved analyses (auth required).
GET/saved-result/{job_id}
Load saved result (auth required).

Interactive docs at http://localhost:8000/docs
